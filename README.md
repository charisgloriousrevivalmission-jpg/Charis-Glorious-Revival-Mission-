🚀 Deployment
This site is deployed on Netlify via drag-and-drop.
To deploy your own copy:
Clone or download this repository
Go to app.netlify.com/drop
Drag the project folder onto the page
Your site will be live instantly
To connect a custom domain, go to Site Settings → Domain Management in Netlify.
🙏 Prayer Request Form
The prayer form uses Netlify Forms — no backend required. Submissions appear automatically in your Netlify dashboard under Forms.
To enable email notifications:
Go to Netlify → Site Settings → Forms → Form Notifications
